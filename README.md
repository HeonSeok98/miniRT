## miniRT
My first RayTracer with miniLibX

C언어와 miniLibX 라이브러리를 이용하여 레이트레이싱을 구현하는 프로젝트입니다.

+ phong lighting(Ambient, Diffuse, Specular)로 구성
+ 평면, 원기둥, 구, 원뿔 구현
+ checkerboard, bump mapping 구현

## 실행하기
```text
make or make bonus
./miniRT ./*.rt or ./miniRT_bonus ./*.rt
```

## 기술스택
C, miniLibX
